# CleanFlow
Scheduling app for cleaning operations — Malta.

Built for short-let property management companies managing 35+ properties, 
restaurant and office cleaning contracts.

## Features
- Auto-scheduler based on cleaner training, availability and weekly hour limits
- Hostaway CSV import for weekly checkout data
- AI assistant (Gemini) for scheduling exceptions in Italian
- When I Work schedule reference export

## Stack
React · Google Gemini API · SheetJS

## Usage
Open the app, load the default template, import Hostaway checkouts, 
generate shifts, review with AI, export to When I Work.
